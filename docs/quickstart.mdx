## Prerequisites

- Python 3.11+ (3.13 recommended)
- [uv](https://docs.astral.sh/uv/) package manager
- [Nebius Token Factory](https://studio.nebius.ai/) API key
- [Tavily](https://tavily.com/) API key

## Installation

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/colygon/accenture-adm-crew.git
    cd accenture-adm-crew
    ```
  </Step>
  <Step title="Create a virtual environment and install dependencies">
    ```bash
    uv venv --python 3.13
    uv sync
    uv pip install chainlit tavily-python
    ```
  </Step>
  <Step title="Configure API keys">
    Create a `.env` file in the project root:

    ```env
    NEBIUS_API_KEY=your_nebius_token_factory_key
    TAVILY_API_KEY=your_tavily_api_key
    ```

    <Info>
      Get your Nebius API key at [studio.nebius.ai](https://studio.nebius.ai/). 
      Get your Tavily API key at [tavily.com](https://tavily.com/).
    </Info>
  </Step>
  <Step title="Launch the web UI">
    ```bash
    .venv/bin/chainlit run app.py
    ```

    Open [http://localhost:8000](http://localhost:8000) in your browser.
  </Step>
</Steps>

## Your first engagement

Once the UI is running:

1. Enter your **company name** when prompted
2. Describe the **type of engagement** (e.g., "Digital Transformation - Cloud Migration")
3. Watch the crew work through each ADM phase
4. Answer any questions the consulting team asks you
5. Browse your deliverables when the engagement completes

## CLI usage

You can also run the crew from the command line without the web UI:

```bash
.venv/bin/python -m accenture_adm_hierarchical_delivery_crew.main run
```

This will prompt you for the engagement details interactively.
