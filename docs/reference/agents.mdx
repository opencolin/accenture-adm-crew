## Agent hierarchy

The crew models Accenture's delivery organization with three tracks:

```
                    Managing Director (Manager)
                           │
          ┌────────────────┼────────────────┐
    Strategy &        Technology        Delivery &
    Consulting          Track           Operations
```

## Manager agent

The Managing Director is the **manager agent** in CrewAI's hierarchical process. It coordinates task execution but does not appear in the worker pool and has no tools.

| Property | Value |
|----------|-------|
| Role | Managing Director at Accenture |
| Goal | Own client relationships and ensure sustained, enterprise-level value |
| Tools | None (manager agents cannot have tools) |
| Delegation | Enabled |
