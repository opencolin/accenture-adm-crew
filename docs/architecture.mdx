## Hierarchical process

The crew uses CrewAI's **hierarchical process** mode. The Managing Director acts as the manager agent — it does not appear in the worker pool but coordinates task execution across all other agents.

Key behaviors:
- The MD **delegates tasks** to the most appropriate agent based on the task description and agent capabilities
- Worker agents can **delegate to each other** when they need expertise from a different role
- **Client-facing agents** (strategy and consulting track) have access to the "Ask Client" tool for human-in-the-loop interaction
