# Accenture ADM Delivery Crew

A multi-agent AI system that simulates Accenture's Application Delivery Method (ADM) lifecycle. Built with [CrewAI](https://crewai.com) and powered by [Nebius Token Factory](https://tokenfactory.nebius.com) inference.

## What it does

The system deploys **11 AI agents** — each representing a distinct Accenture delivery role — to collaboratively execute a full enterprise engagement lifecycle. A Managing Director agent orchestrates the crew hierarchically, delegating work across strategy, consulting, architecture, development, testing, and deployment phases.

## How it works

1. **Client provides input** — company name and engagement type (e.g., "Digital Transformation", "Cloud Migration")
2. **Managing Director orchestrates** — delegates tasks across the agent hierarchy
3. **Agents research and collaborate** — using web search, site scraping, and inter-agent delegation
4. **Client-facing agents ask questions** — when they need clarification on requirements, priorities, or constraints
5. **12 deliverables are produced** — one per ADM phase, saved as markdown files
